## Overview
AURA is an AI-powered emergency response system that detects medical emergencies using real-time monitoring and automatically dispatches emergency services.

## Features
- AI-based collapse detection
- Inactivity detection
- Emergency validation engine
- Nearest hospital identification
- Ambulance dispatch API integration
- SMS/Email alert system
- Real-time incident logging

## Tech Stack
- Python (Flask)
- Turbotic AI (Automation Engine)
- Render (Backend Hosting)
- GitHub (Version Control)

## Deployment
Backend deployed on Render:
"https://aur-dispatch.onrender.com/dispatch"


## How It Works
1. AI detects emergency event.
2. Validation engine confirms severity.
3. System fetches nearest hospital.
4. Emergency dispatch API is triggered.
5. Notifications sent to contacts.
